# LeetCode Solutions

A curated collection of LeetCode problem solutions organized by topic. Each problem folder includes a concise README with the problem summary, approach, and complexity, along with a clean Python implementation.

## Structure
- Topic folders: arrays, strings, linked-list, stack, queue, trees, graphs, dynamic-programming, greedy
- Each problem folder contains:
  - README.md
  - solution.py

## How to Use
1. Browse a topic folder to find a problem.
2. Read the problem README for the approach and complexity.
3. Open solution.py for the implementation.

## Notes
- Solutions focus on clarity and efficiency.
- Additional languages can be added on request.
